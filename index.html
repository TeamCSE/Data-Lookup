📖 User Instructions

To process large files (1GB+) without memory errors, please follow these steps:

  1. Convert to CSV: Open your Excel file, go to File > Save As and select CSV (Comma Delimited).
  2. Select File: Click "Choose File" and pick the .csv version.
  3. Enter Keywords: Paste your codes into the box. You can separate them with spaces, commas, or new lines.
  4. Run Search: Click "Run Search". The system will read the file line-by-line to save your RAM.

    5. Note: For efficient and faster data search, i suggest to edit the CSV file and leave only the following column: Session ID, Transaction ID, Skill Name
